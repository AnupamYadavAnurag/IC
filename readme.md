# 🍽️ India Canteen - Restaurant Menu Website

A modern, responsive restaurant menu website developed for **India Canteen**. This project showcases a premium digital menu with an attractive UI, smooth navigation, and a mobile-friendly design.

## 🌟 Features

- 🍔 Three beautifully designed menu pages
- 📱 Fully Responsive Design
- 🎨 Modern & Premium UI
- ⚡ Fast Loading
- 🧭 Previous & Next Navigation
- 📥 Download Menu Button
- 🍽️ Organized Food Categories
- 📞 Contact Information
- 🌙 Clean Layout
- 💻 GitHub Pages Ready

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
